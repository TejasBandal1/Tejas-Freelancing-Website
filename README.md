# Tejas Bandal Portfolio

Premium single-page freelancing portfolio built with React, Tailwind CSS, and Framer Motion.

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
